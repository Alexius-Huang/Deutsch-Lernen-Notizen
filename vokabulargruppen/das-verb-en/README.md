# Das Verb / -en

* kommen / kommend / **ge**kommen \(come\)
  * **be**kommen \(to receive; to get\)
  * [ ] **an**kommen; kommen **an** \(to arrive; to reach\)
  * [ ] **um**kommen; kommen **um** \(to die\)
  * [ ] **aus**kommen; kommen **aus** \(+ mit\) \(to get along with sb. well\)
